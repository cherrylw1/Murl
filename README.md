# Murl

Murl is a local-first, native-macOS multi-agent web-research harness.

## Development and Testing

To install dependencies:

```bash
pnpm install
```

To run tests:

```bash
pnpm test
```

To typecheck:

```bash
pnpm typecheck
```

To lint:

```bash
pnpm lint
```

To format code:

```bash
pnpm format
```
